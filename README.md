# Разработка модели глубокого обучения для прогнозирования продуктивности сельскохозяйственных растений по генетическим маркерам

## 📌 Описание
Краткое описание работы: трансформерная модель с вниманием на уровне SNP и реконструкционной регуляризацией для прогнозирования фенотипов нута, риса и рапса.

## 🏗️ Структура проекта
Описание основных папок (см. выше).

## ⚙️ Установка и запуск
- Python 3.8+
- Установка зависимостей: `pip install -r requirements.txt`
- Запуск обучения: :)

## 📊 Данные
Источники данных:
- Нут — данные ВИР (Соколькова и др., 2020)
- Рис — Lu et al., 2015 (BMC Genomics)
- Рапс — CropGS-Hub (Chen et al., 2024; Wu et al., 2019)
